TLTB Mini - Simple WiFi AP Test Instructions
==============================================

1. UPLOAD THE FIRMWARE:
   - Use: "C:\Users\colla\.platformio\penv\Scripts\platformio.exe run --target upload --environment esp32-s3-devkitc-1"

2. POWER ON THE ESP32 and check serial output:
   - Should show: "AP started: TLTB-Mini-XXXXXX (WPA2 password: TLTB1234)"
   - Should show: "AP IP: 192.168.4.1"

3. CONNECT TO WIFI:
   - Look for WiFi network: "TLTB-Mini-XXXXXX" (password: TLTB1234)
   - Connect to it from your phone/computer

4. TEST ACCESS:
   - Open browser and go to: http://192.168.4.1
   - You should see the TLTB Mini interface

5. IF IT DOESN'T WORK:
   - Check serial output for errors
   - Try going to: http://192.168.4.1/info for diagnostics
   - Make sure you're connected to the TLTB-Mini WiFi network
